# Customer Analytics Dashboard | Power BI Project #

This repository contains an end-to-end data analytics project focused on analyzing customer data and building an interactive Power BI dashboard.
The project demonstrates the complete analytics workflow — from data cleaning and analysis to visual storytelling and business insights.


# Project Overview
The goal of this project is to analyze customer-related data and present key business metrics using Power BI.
It helps stakeholders quickly understand:

1. Customer distribution
2. Revenue and pricing metrics
3. Key performance indicators (KPIs)
4. Overall business trends


# Tools & Technologies Used
1. Python – Data cleaning and preprocessing
2. SQL – Data querying and analysis
3. Power BI Desktop – Dashboard creation and visualization
4. CSV / Excel – Raw data sources


# Dashboard Highlights

The Power BI dashboard includes:
1. Total Number of Customers
2. Revenue / Price Metrics
3. Key KPIs using Card Visuals
4. Interactive Filters & Slicers
5. Clean layout for easy interpretation


# Project Workflow

Data Collection: Raw customer data collected in CSV format

Data Cleaning (Python):
1. Removed null values
2. Standardized columns
3. Prepared clean dataset for analysis

Data Analysis (SQL):
1. Derived business metrics
2. Aggregated customer and pricing data

Data Visualization (Power BI):
1. Created KPI cards
2. Designed interactive dashboard
3. Applied formatting and layout best practices


# Key Insights

Some insights derived from the dashboard include:
1. Total customer count and trends
2. Average pricing metrics
3. Overall business performance indicators
4. Data-driven insights useful for decision-making
